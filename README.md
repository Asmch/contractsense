# 🚀 Overview

ContractSense is an AI-powered Contract Intelligence Platform designed to make legal agreements easy to understand for everyone.

Instead of reading pages of complex legal language, users simply upload a contract and receive:

- Plain-English explanations
- Hidden risk detection
- Actionable recommendations
- Negotiation guidance
- AI-powered contract chat
- Professional contract reports

Our mission is simple:

> **Help people make confident decisions before signing any agreement.**

# 🛠 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/contractsense.git
```

Navigate to the project

```bash
cd contractsense
```

Install dependencies

```bash
npm install
```

Create an environment file

```env
NEXTAUTH_SECRET=

MONGODB_URI=

GOOGLE_GEMINI_API_KEY=

CLOUDINARY_CLOUD_NAME=

CLOUDINARY_API_KEY=

CLOUDINARY_API_SECRET=
```

Run the development server

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
